矩阵属性
''''''''''''''''''''''''''

``numpy.ndarray`` 的属性:

:ndim:      维度个数
:shape:     数组形状
:size:      元素个数
:dtype:     元素类型

.. code:: python
  
  assert arr.ndim == 2
  arr.shape
  arr.size
  arr.dtype
